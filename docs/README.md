# Deploy ThatNightSky

This page describes how to deploy ThatNightSky to various environments.

## Deployment options

| Guide | Description | Status |
|-------|-------------|--------|
| [Run in Docker](setup-docker.md) | Build and run locally or on a home server using Docker Compose | Stable |
| [Set Up Cloudflare Tunnel](setup-cloudflare.md) | Expose the Docker deployment to the internet without port forwarding | Stable |
| [Deploy to Streamlit Community Cloud](setup-streamlit-cloud.md) | One-click deployment on Streamlit's free hosting platform | ⚠️ Work in progress |
